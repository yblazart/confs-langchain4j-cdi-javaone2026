📋 Configuration (in microprofile-config.properties):

🔸 Ollama (local):

Then run: ollama pull llama3.2-vision:11b

🔸 OpenAI (cloud):
vision-model.class=...OpenAiChatModel
vision-model.config.model-name=gpt-4o
vision-model.config.api-key=${OPENAI_API_KEY}
Env variable: export OPENAI_API_KEY=sk-...

🔸 GROQ (cloud, fast):
vision-model.class=...OpenAiChatModel
vision-model.config.base-url=https://api.groq.com/openai/v1
vision-model.config.model-name=meta-llama/llama-4-scout-17b-16e-instruct
vision-model.config.api-key=${GROQ_API_KEY}
Env variable: export GROQ_API_KEY=gsk_...
